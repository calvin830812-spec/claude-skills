若素材不足，詢問：

## Step 1：解析報價單

讀取每份報價 PDF，記錄：

| 欄位 | 說明 |
|------|------|
| 廠商名稱 | 公司全名 |
| 負責人 | 姓名（用於說明欄備查） |
| 工程項目清單 | 各項名稱與金額 |
| 稅前小計 | 不含稅總額 |
| 稅金（5%） | 應付稅額 |
| 含稅合計 | 最終報價 |
| 報價日期 | 民國年月日 |
| 備註 | 含特殊範圍說明（如「含外圍 20 公尺」、「含違章調查」等） |

## Step 2：費用計算規則

**議價後費用計算方式（重要）：**

```
議價後稅前小計 = 原始稅前小計 − 議價金額
最終含稅費用   = 議價後稅前小計 × 1.05
```

> ⚠️ 注意：議價金額從「稅前小計」扣除後再計稅，**不是**從含稅合計直接扣減。

若使用者只說「議價 X 萬」，預設從稅前小計扣除。

### 說明欄結構

說明欄分三大段：

**開頭段（案由）**

```
為推動「{完整案名}」，需委託{服務類型}，{交付成果說明}，{用途說明}。

爰依比價程序取得三家廠商報價，茲就各家報價內容比較分析如下：
```

**一、比價彙總（表格）**

| 廠商 | 報價金額（含稅） | 備註 |
|------|----------------|------|
| A 公司 | NT$xxx,xxx | 前期執行廠商（如有） |
| B 公司 | NT$xxx,xxx | 含 XXX 額外範圍（如有）|
| C 公司 | NT$xxx,xxx | |

**二、比較分析**

分析點依實際差異而定，常見分析維度：

- **(一) 服務範疇**：說明三家是否涵蓋同等核心服務項目
- **(二) 報價金額**：指出最高報價的溢付點，解釋為何不選最低報價（如有理由）
- **(三) 場域熟悉度 / 延續性**：如傾向廠商有前期合作或場域經驗，在此說明

> 注意：只寫有實質差異的維度。若三家服務範疇相同，不要硬湊差異點。

**三、議價結果與建議**

```
綜上，建議委託{廠商名}執行。經溝通議價，廠商同意讓步新台幣{文字金額}元整，
最終執行費用 NT${最終費用}，較原始最低報價{最低廠商名}（NT${最低原報價}）
仍低 NT${差額}，兼顧成本控制與成果品質。

委託廠商：{廠商名}（負責人：{姓名}）
最終費用：NT${最終費用}（含稅，含{主要交付項目}）
　　預計於「{折扣所在項目}」減{議價金額}元，與簽呈附件資料所列金額有出入，特此說明。

附件一：{廠商A}報價單
附件二：{廠商B}報價單
附件三：{廠商C}報價單

敬請　核示。
```

## 參考案例

**信義364測量案（2026-05-11）**

- 案名：臺北市大安區大安段一小段364地號等19筆土地都市更新計畫
- 委外項目：現況測量及建物調查工程
- 三家報價：群測 NT$168,000 / 垣華 NT$184,275 / 緯達 NT$210,000
- 傾向廠商：垣華工程有限公司（前期已執行地形測量）
- 議價：NT$20,000（從稅前小計 $175,500 扣除）
- 最終費用：($175,500 − $20,000) × 1.05 = **NT$163,275**
- 主旨：`信義364 簽呈 1150511 莊穎霖 委託垣華工程有限公司辦理都市更新測量工程費用乙案`
